# Roadmap

The Argo Rollouts roadmap is maintained in [Github Milestones](https://github.com/argoproj/argo-rollouts/milestones) on the Github repository.

## Release Process

### Minor Releases (e.g. 1.x.0)

A minor Argo Rollouts release occurs four times a year, once every three months. Each release includes all features
of the master branch (at the point in time) and any additional release notes or points of attention.

Dates may be shifted slightly to accommodate holidays. Those shifts should be minimal.

### Patch Releases (e.g. 1.4.x)

Argo Rollouts patch releases occur on an as-needed basis. They contain mostly security fixes for important vulnerabilities.
